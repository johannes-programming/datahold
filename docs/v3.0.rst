Overview
--------

This project wraps common mutable datastructures for inheritance with modification.

Installation
------------

To install ``datahold``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install datahold

Typing
------

The typing conforms strictly to ``mypy``.

Features
--------

This abstract base class is the ancestor of all other classes in ``datahold``.

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

``__gt__(other: Any, /) -> Any``
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

``__le__(other: Any, /) -> Any``
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This magic method implements ``self <= other``.

``__lt__(other: Any, /) -> Any``
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This magic method implements ``self < other``.

``__ne__(other: Any, /) -> Any``
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This magic method implements ``self != other``.

``data: Hashable``
^^^^^^^^^^^^^^^^^^

This property is abstract.

``class datahold.base.BaseDataCollection.BaseDataCollection(data: Any, /)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

``__contains__(other: Hashable, /) -> bool``
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This magic method implements ``other in self``.

.. code-block:: python

    def __contains__(self: Self, other: Hashable, /) -> bool:
        return other in self.data

``__iter__() -> Iterator[Item]``
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This magic method implements ``iter(self)``.

.. code-block:: python

    def __iter__(self: Self, /) -> Iterator[Item]:
        return iter(self.data)

``__len__() -> int``
^^^^^^^^^^^^^^^^^^^^

.. code-block:: python

    def __len__(self: Self, /) -> int:
        return len(self.data)

``data: HashableCollection[Item]``
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This abstract property is at the center of this class and of its descendants.
